Sýnidæmi um hvernig hægt er að hafa fleiri en eitt pull request template 

## Leiðbeiningar

í .github foldernum

1. Bættu við skránni pull_request_template.md með eftirfarandi leiðbeiningum 

`[Tegund A](?expand=1&template=tegundA_template.md)`<BR>

`[Tegund B](?expand=1&template=tegundB_template.md)`

Þessi texti birtist í hvert skipti sem gert er Pull Request. Höfundurinn velur þá annað hvort (eða úr fleirum) templates. 

2. Bættu við foldernum PULL_REQUEST_TEMPLATE

3. Í foldernum PULL_REQUEST_TEMPLATE bættu við tveimur (eða fleiri ) template skrám 

tegundA_template.md
tegundB_template.md 


## Þetta dæmi 

Í þessu dæmi er ANNAD_PR_Template.md fyrir tegundA_template.md og
FR_PR_template.md fyrir tegundB_template.md
